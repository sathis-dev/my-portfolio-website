Before implementing any component, verify:

- [ ] Background color matches element type (card vs input vs button)
- [ ] Icon uses solid fill, NOT background or gradient
- [ ] Text color matches hierarchy (heading vs body vs accent)
- [ ] Border opacity correct for element (0.06 vs 0.08 vs 0.12)
- [ ] Shadow applied only to appropriate elements (cards, buttons)
- [ ] Gradient only on buttons and hero section
- [ ] Hover states include transform (lift) where appropriate
- [ ] Focus states use purple accent colors
- [ ] Border radius matches component size
- [ ] Spacing uses consistent scale (0.5rem, 0.75rem, 1rem, etc.)
