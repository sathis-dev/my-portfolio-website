# Project Images

Place your project images in the appropriate folders.

## Directory Structure

```
projects/
├── silent-help/
│   └── thumbnail.jpg (16:9 ratio, recommended 1920x1080)
├── brimston/
│   └── thumbnail.jpg (16:9 ratio, recommended 1920x1080)
└── airsense/
    └── thumbnail.jpg (16:9 ratio, recommended 1920x1080)
```

## Image Requirements

- **Format:** JPG, PNG, or WebP
- **Aspect Ratio:** 16:9 (landscape)
- **Recommended Size:** 1920x1080px
- **Max File Size:** < 500KB for optimal performance

## Naming Convention

- Thumbnail for list/grid: `thumbnail.jpg`
- Additional images for gallery: `image-1.jpg`, `image-2.jpg`, etc.
